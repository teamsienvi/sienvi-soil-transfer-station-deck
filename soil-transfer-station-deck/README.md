# AI-Powered Soil Transfer Station Deck

This is a lightweight animated pitch deck built as a static site.

## Open locally

From this folder, run:

```sh
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173
```

## Controls

- `Next` / `Previous` buttons
- Arrow keys
- Spacebar
- Swipe on touch devices
